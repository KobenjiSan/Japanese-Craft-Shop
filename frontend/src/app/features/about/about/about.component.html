What is Komorebi?

木漏れ日 (Komorebi) Japanese Craft Shop is a demo e-commerce catalog built to showcase full-stack development skills while simulating the experience of browsing and collecting authentic Japanese crafts. Designed for craft enthusiasts and design-minded users, it demonstrates an end-to-end system including authentication, product CRUD, admin dashboard, file uploads, and dynamic filtering/sorting. The project was built entirely from scratch, combining Angular 20 + Tailwind CSS on the frontend with an ASP.NET Core Web API backend following CQRS + MediatR + DTOs, secured with JWT, and powered by MongoDB. While fictional, the app weaves in the aesthetic of komorebi (sunlight filtering through leaves) blending cultural inspiration with technical execution.

Origin & Intent

The name 木漏れ日 (Komorebi) comes from Wim Wenders' film Perfect Days, a movie I first watched before my study abroad in Japan and then carried with me during that time. Its quiet focus on light through trees, daily rituals, and contentment in ordinary life mirrored what I experienced in Japan: the mindfulness and joy found in small moments. That sense of stillness and presence inspired the way I shaped this project as not just an app, but as a digital space meant to feel intentional and unhurried.

Highlights

  • Full-stack architecture: Angular + Tailwind CSS frontend with ASP.NET Core Web API backend, secured with JWT and powered by MongoDB
  • CQRS + MediatR pattern: Clean separation of commands / queries for maintainable, scalable backend logic
  • Authentication & Authorization: Role-based system with JWT tokens, supporting Admin vs Visitor flows
  • Admin Dashboard: Product CRUD (create, update, delete), file uploads, and category management
  • Dynamic product browsing: Filtering, searching, and sorting integrated for a smooth catalog experience
  • Modern UI/UX: Responsive design with Tailwind, inspired by Japanese craft aesthetics and the concept of komorebi.
